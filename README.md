
#  Car Rental Landing Page

Responsive landing page for a modern car rental business.

Built as a front-end portfolio project to demonstrate clean UI structure, responsive layout design, and basic interactive form logic using vanilla JavaScript.

---

## Features

- Responsive dark UI design
- Modern layout with CSS Grid & Flexbox
- Interactive booking form with instant price estimation
- Mobile navigation menu
- Clean file structure (HTML / CSS / JS)

---

## Tech Stack

- HTML5
- CSS3 (custom properties, responsive design)
- Vanilla JavaScript
